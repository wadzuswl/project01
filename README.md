# 🌱 Project01 | 대전충남녹색연합 홈페이지 리뉴얼

> **환경 단체의 메시지를 한눈에, 행동으로 이어지게 만드는 홈페이지 리뉴얼**

환경 단체의 정체성과 가치를 명확하게 전달하고  
사용자가 자연스럽게 참여와 후원으로 이어질 수 있도록 개선한  
**대전충남녹색연합 공식 홈페이지 리뉴얼 프로젝트**입니다.

## 🔍 프로젝트 소개

대전충남녹색연합은 대전·충남 지역을 중심으로  
환경 보호와 시민 참여 활동을 전개하는 **비영리 시민 단체**입니다.

기존 홈페이지는 정보 전달 위주의 구성으로  
단체의 메시지와 신뢰도, 후원 유도 측면에서 아쉬움이 있었습니다.

본 프로젝트는 이러한 문제를 개선하여  
**환경 단체의 가치가 직관적으로 전달되는 웹 경험**을 만드는 것을 목표로 진행되었습니다.

---

## 🎯 프로젝트 목표

- 환경 단체의 핵심 메시지를 메인 페이지에서 명확하게 전달
- 과도한 색상 사용을 줄여 자연 친화적인 디자인 구현
- 정보 구조 재정비를 통한 가독성과 이용 효율 개선
- 후원 버튼 및 CTA 영역 강화를 통한 사용자 행동 유도

---

## 👥 타겟 사용자 분석

| 사용자 유형 | 비율 | 특징 |
|------------|------|------|
| 일반 방문자 | 65% | 모바일 접속 비중 높음 |
| 회원 / 후원자 | 20% | 단체 소식 및 후원 정보 관심 |
| 활동가 / 운영자 | 10% | PC 사용 위주 |
| 연구자 / 언론 | 5% | 자료 탐색 목적 |

➡ **모바일 중심 UI와 정보 접근성 개선**에 중점

## ❗ 기존 사이트 문제점

### 전반적인 문제
- 초록색 컬러의 과도한 사용으로 시각적 피로도 증가
- 정적이고 평면적인 레이아웃
- 굵은 폰트 사용으로 가독성 저하
- 후원 유도 영역의 시인성 부족
- 단체 신뢰도를 보여주는 콘텐츠 부재

### 메인 페이지
- 메인 비주얼이 단순 소식 노출에 그침
- 정보·소식 영역 비중 과다로 스크롤 부담 발생
- 콘텐츠 분류 체계 부족

### 푸터
- 작은 글자와 강한 컬러 대비로 가독성 저하
- SNS 아이콘 위치 인지 어려움

---

## 🔧 리뉴얼 방향 & 해결 방안

- **핵심 메시지를 중심으로 한 메인 비주얼 구성**
- 콘텐츠를 목적별로 재구성하여 정보 탐색 효율 향상
- 후원 버튼을 명확하게 배치하여 사용자 행동 유도
- 여백과 카드형 UI를 활용해 안정적이고 신뢰감 있는 인상 제공

---

## 🎨 스타일 가이드

### Color
- Green 계열을 메인 컬러로 사용하되 채도 조절
- Yellow 컬러로 포인트 강조
- White / Gray 컬러로 가독성 보완

### Typography
- 제목과 본문에 명확한 위계 적용
- 과도한 굵기 사용을 지양하여 읽기 편한 구조 설계

### UI
- 카드형 레이아웃과 섀도우 활용
- 일정한 spacing 시스템으로 정돈된 화면 구성
